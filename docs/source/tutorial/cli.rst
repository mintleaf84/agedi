Command Line Interface
======================
After installing AGeDi try running

.. code-block:: console
   agedi --help

The will look like
.. click:: agedi:agedi
   :prog: agedi
   :nested: short

The CLI exposes the two main functionalities namely training and
sampling the model. To inspect the possibilities within try

.. code-block:: console
   agedi train --help

and

.. code-block:: console
   agedi sample --help
